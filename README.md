
# 🧠 Testcore.uz – Online Mock Test Platform

Testcore.uz is a modern web and mobile platform designed to help **university applicants**, **Presidential School candidates**, and **students preparing for Olympiads** practice real-style mock exams. The platform uses AI to analyze results, provide smart feedback, and even generate new test questions.

---

## 📌 Key Features

- 🎯 **Targeted Mock Tests**  
  Choose from categories like:
  - Presidential Schools
  - DTM (State Test Center)
  - Universities & Faculties

- 🤖 **AI-Powered Feedback**  
  After completing tests, users receive AI-generated analysis of their performance, including question-by-question breakdowns.

- 🧪 **Create Tests From PDFs or Text (AI-Generated)**  
  Teachers or students can upload content and get question sets automatically created using AI.

- 🧑‍🏫 **Teacher Panel**  
  - Upload DOCX/PDF files and convert them to tests
  - Track student test results
  - Generate test links and QR codes (like Kahoot.it)

- 🏆 **Olympiad Mode**  
  Time-limited competitions with single-start schedules and entry payments.

- 📊 **User Profiles & Statistics**  
  Each user has a profile showing test history, rankings, subscription status, and more.
